# Authority Labs

[![Software License][ico-license]](LICENSE.md)

Authority Labs API Library

## Install

Via Composer

``` bash
$ composer require aliasproject/authority-labs
```

## Usage

``` php
$authorityLabs = new AliasProject\AuthorityLabs();
```
